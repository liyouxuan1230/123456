
# 甘肃高速路政安全生产管理系统

本系统是专为**甘肃省高速公路路政执法总队安全科**开发的管理平台。

## 如何上线微信小程序？

由于本项目是基于 React 的单页 Web 应用，最稳定的上线方式是：

1.  **域名准入：** 准备一个 HTTPS 域名。
2.  **业务域名配置：** 在微信小程序后台添加该域名为“业务域名”。
3.  **代码封装：** 在小程序中使用 `<web-view>` 组件加载本应用。

**详细步骤已内置在软件中：**
运行软件后，点击左侧菜单栏底部的 **“部署指引”**，即可查阅手把手的上线教学。

## 技术栈
- React 19 + Tailwind CSS
- Lucide React 图标库
- Recharts 数据可视化
- Google Gemini 3 AI 智能分析（用于安全建议生成）
